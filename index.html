Game Manual

Welcome to the Memory game 1.0! Here's how to play:

1. The game will start with a single color shown in the center.

2. Remember the color and click/tap the same color to continue.

3. The sequence will get longer as you progress.

4. If you make a mistake, the game will end.

5. Try to beat your high score!
